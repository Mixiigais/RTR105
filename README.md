### Pārvietošanas failu sistēma:  
**cd** **.** - action on the same spot
**cd** **..** - action backwards
**cd map_name/**  
**cp file_name** - copy file  
**mv file_name** - move file 
**rm file_name** - remove  
**cat/more/less file_name** - view file content
